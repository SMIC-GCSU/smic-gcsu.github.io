About the Software

The SMIC Portfolio Analysis Tool is a comprehensive desktop application designed for tracking and analyzing the Student Managed Investment Committee's portfolio performance. It provides real-time portfolio valuation, sector allocation analysis, performance metrics, and detailed reporting.

Features

  • Real-time portfolio value tracking
  • Sector allocation analysis with drift visualization
  • Performance comparison with S&P 500 benchmark
  • ETF vs Individual Stocks breakdown by sector
  • Interactive charts and visualizations
  • Comprehensive statistics and reporting
  • Export capabilities for reports and data

Installation

Windows: Download the .exe file and run it directly. No installation required.

macOS: Download the .zip file, extract it, and move the .app to your Applications folder. You may need to allow the app in System Preferences > Security & Privacy on first launch.
